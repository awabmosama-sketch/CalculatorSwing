# Simple Calculator (Java Swing)

A functional calculator application developed using **Java Swing**. This project represents my first experience with Java GUI development, focusing on event handling, layout management, and component design.

## Features
* **Arithmetic Operations:** Addition, Subtraction, Multiplication, Division, Modulo, and Exponentiation.
* **Responsive UI:** Built using `GridLayout` for a structured and clean interface.
* **Error Handling:** Robust input validation to handle division by zero and invalid numeric formats.
* **Custom Branding:** Window customization with a custom application icon.

## Technologies Used
* **Language:** Java
* **Framework:** Swing (javax.swing)
* **IDE:** IntelliJ IDEA

## Getting Started
To run this project, ensure you have the Java Development Kit (JDK) installed. Simply compile and run the main class in your preferred Java IDE.

## About
This is my first project utilizing Java Swing for graphical user interfaces. It provided a great opportunity to learn about `JFrame`, `ActionListener`, and the fundamental principles of building desktop applications in Java.
